Приложение для размещения фотографий в общем фотоальбоме

Ссылка на проект - https://4th-king.github.io/mesto-production/

Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка проекта
npm run build

# Деплой (запускается автоматически при push в main)
git push origin main
#Или
npm run deploy
